# Comprehensive-Workforce-Attrition-Analysis-Dashboard
Comprehensive Workforce Attrition Analysis Dashboard

This interactive dashboard offers advanced analytics to enhance workforce insights:

✔️ Interactive Slicers: Filter data by Phase, Process, Sub Process, Year, Month, Gender, and Tenure.
✔️ Attrition Analysis: Semi-circle charts for Total Attrition %, YTD Attrition %, and Annualized Attrition %.
✔️ Attrition Breakdown: Doughnut chart for Voluntary & Involuntary counts, with total attrition displayed centrally with gender-Based .
✔️ Monthly Trends.
